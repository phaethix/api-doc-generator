## June 2026

| Status | Feature | Description |
|--------|---------|-------------|
| 📋 | **Multiple LLM providers** | Add built-in support for OpenAI, Anthropic, and Ollama |
| 📋 | **Swagger 2.0 import** | Support importing legacy Swagger 2.0 specs |
| 📋 | **OpenAPI file upload** | Upload `.json`/`.yaml` files in the UI instead of pasting |
| 📋 | **YAML output format** | Generate docs in YAML alongside Markdown/HTML/JSON |
| 📋 | **Frontend unit tests** | Vitest + React Testing Library |
| 📋 | **Response examples in docs** | Render example request/response bodies in generated output |

| Status | Feature | Description |
